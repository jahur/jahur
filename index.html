AI Palette

https://www.aipalette.com/

Client overview

AI Palette helps the FMCG companies with their new product innovation. We use Artificial Intelligence & Machine Learning to help FMCG companies create consumer winning products by identifying food trends in real time.

Project overview

At the initial stage of the project I showed a few wordpress themes form themeforest. They chose one design built in Detox theme. Then they told me, they will give me a design in Adobe XD. Unfortunately they did not use any of the elements available in Detox theme.

I started everything from scratch using Elementor page builder according to their design.

Key points of the project

  • Complete website

    Designed each of the pages of this website.

  • Ajax filter

    Developed a blog post filter using ajax which was not available in the theme.

  • Home page featured post

    Developed an element which can fetch blog posts based on their selection.

  • Options to Redux theme option page

    Edited the theme option file to add some extra fields.

  • Custom post template

    Created two new post templates according to their need. They had two separate designs for the templates. I added custom fields using ACF (Advanced Custom Fields) plugin.

KAN Property

https://kanproperty.com.au/

Client overview

Kerry-Anne Nielsen brings experience and talent with her wealth of knowledge having worked in the Real Estate industry for over 20 years. She has survived all the real estate markets over the years and is adaptive to changing conditions. Her results for clients ensures Kerry-Anne stays top of mind and is the go-to Industry Professional people turn to when they want honest reliable advice.

Project overview

Client chose a theme from themeforest called Residio. Residio is Elementor based theme. I desined all pages according to their need.

Our cient used a CRM called AgentBox. AgentBox uploads XML files to our server when new property added or updated in their system. I process those XML files and wrote a script which imports properties in WordPress.

Key points of the project

  • Designed all pages including customization in some parts of the theme
  • Import script which imports properties to WordPress
  • Created cron jobs to automate the import
Plugin Development, WooCommerce, REST API

ekshop Sync

Client overview

ekshop is a Bangladesh government owened ecommerce platform which unifies all ecommerce entities in Bangladesh. All ecommerce businesses can showcase their products.

Project overview

My responsibility was to create a WordPress plugin which can communicate with ekShop and sync WooCommerce products. I was given API endpoints and the structure of their products. I mapped all types of WooCommerce products, wrote script which can send all published products to ekShop. It generates a report which products are imported successfully and which are not. If product is updated, the status of product will be changed so that it can be recognized.

Automatic sync is performed in every 48 hours using WPCron.

Key points of the project

  • Map WooCommerce product data according to ekshop product data
  • Setup connection to ekshop server to send the product data
  • Generate report after completion of sync
  • Manual sync button in WooCommerce product list
